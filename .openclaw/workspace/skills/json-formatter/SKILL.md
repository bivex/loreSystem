# json-formatter

Базовый скил для форматирования вывода в JSON.

## Правила

1. **Строгий JSON**: Вывод всегда валидный JSON
2. **UUID для id**: Каждая сущность получает уникальный UUID
3. **Структура**: Соблюдай loreSystem схему для каждого типа сущности
4. **Массивы**: Множественные значения в массивах `[]`
5. **Вложенность**: Связанные сущности могут быть вложены или ссылаться по id

## Формат

```json
{
  "entity_type": {
    "id": "uuid",
    "name": "Entity Name",
    "field1": "value1"
  }
}
```
